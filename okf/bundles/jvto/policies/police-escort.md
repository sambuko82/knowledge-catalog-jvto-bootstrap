# Overview

For large groups (typically around 18 or more travelers), JVTO can
coordinate an official traffic-police escort on certain road segments where
the relevant Traffic Police unit approves it. This is a coordination service
subject to authority approval, not a default inclusion. See the travel-guide
page for how it works and any group incentives.
