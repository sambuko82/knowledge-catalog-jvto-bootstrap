# Overview

Every JVTO package includes private transport, a dedicated driver and
guide(s), all entrance fees and permits, drinking water, full pick-up to
drop-off assistance, and meals as stated (at least breakfast). Bromo tours
add the private 4WD jeep; Ijen tours add gas masks and trekking poles (and
health-screening coordination where BBKSDA requires it); Bali-origin tours
add the ferry crossing.

Typical exclusions: flights, visas, travel insurance, tips, and personal
expenses. See the inclusions/exclusions page for the full list.
