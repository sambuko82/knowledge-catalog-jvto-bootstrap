# Overview

JVTO runs private packages from both Surabaya and Bali. The destinations are
the same East Java volcanoes and waterfalls; what changes is the logistics —
where you begin, the direction of travel, and the first transfer time. There
are more routes from Surabaya, while Bali-origin routes add a ferry crossing.
Choose the origin that fits your flights and timeline.
