# Claim

JVTO states that Ijen night-hike packages include the pre-hike health screening,
coordinated as part of the tour. A health certificate is mandatory for every
guest before Kawah Ijen crater entry — the East Java conservation authority
(BBKSDA) sets this requirement, with Surat Edaran SE.1658/KSA.9/2024 cited as
the supporting regulatory authority. BBKSDA sets the rule, not JVTO; JVTO
coordinates the clinic workflow so guests can satisfy it.
